# 大厂面试题速查卡

这是一个纯静态网页，适合直接托管到 GitHub Pages。

## 你后面怎么用

1. 每次你继续问我新的面试题，我就把新的问答追加到 `data.js` 里。
2. 页面会自动把这些题渲染成卡片。
3. 你可以通过搜索框和标签快速索引，不用再翻聊天记录。

## 本地打开

直接双击 `index.html` 就能看。

## 上传到 GitHub Pages

1. 新建一个 GitHub 仓库。
2. 把当前目录文件上传上去。
3. 打开仓库 `Settings`。
4. 找到 `Pages`。
5. 在 `Build and deployment` 里选择 `Deploy from a branch`。
6. 分支选择 `main`，目录选择 `/ (root)`。
7. 保存后，等几十秒到几分钟，就会生成一个公开链接。

## 后续追加数据的格式

在 `window.INTERVIEW_QA` 数组里，按下面结构新增一个对象即可：

```js
{
  id: "Q007",
  category: "算法",
  tags: ["数组", "双指针"],
  question: "什么是双指针？",
  answer: [
    "先写一句定义。",
    "再写典型场景。",
    "最后写复杂度和注意点。"
  ]
}
```
